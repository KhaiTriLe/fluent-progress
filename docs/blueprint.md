# **App Name**: Fluent Progress

## Core Features:

- Practice Timer: Start/stop timer to track speaking practice duration, displaying real-time elapsed time.
- Progress Tracking: Record daily practice time, total practice time, and current streak in a structured and persistent way.
- Sentence Template Library: Provide a categorized collection of English sentence templates covering various topics for users to select from.
- Practice Sentence Selection: Allow users to choose sentences from the template library for inclusion in their practice session.
- Practice counter: Allow users to count the number of times a sentence is used in practice. This increments a numeric count tied to each template sentence.
- Data Import/Export: Enable users to export all application data (practice times, sentence selections, counters, and streak) into a file, and import it back into the application.

## Style Guidelines:

- Primary color: HSL values of 210, 60%, and 50% which converts to a hex value of #3399CC, suggesting a professional, serene tone conducive to learning.
- Background color: HSL values of 210, 20%, and 95% which converts to a hex value of #F0F8FF for a calming and unobtrusive backdrop.
- Accent color: HSL values of 180, 50%, and 40% which converts to a hex value of #339999 for highlighting interactive elements.
- Body text and Headline font: 'PT Sans', a humanist sans-serif that blends modernity and approachability.
- Mobile-first, responsive design with clear, concise elements for easy navigation on smaller screens.
- Simple, clean icons to represent categories and actions.
- Subtle transitions for user interface elements to provide feedback.
